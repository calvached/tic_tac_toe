# Tic Tac Toe
Basic game of Tic Tac Toe

## Ruby version
Built in Ruby 2.1.1p76

## Configuration
Clone the repo then navigate to the bin folder and start the game with:
```
ruby tictactoe
```

## How to run the test suite
Navigate to the main directory and execute:
```
rspec -c
```
